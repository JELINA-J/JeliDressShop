 👗 JeliDressShop

JeliDressShop is a modern e-commerce web application for women's fashion. It provides a seamless online shopping experience with responsive design, product browsing, shopping cart functionality, authentication, and an interactive chatbot for customer assistance.


 📌 Project Overview

JeliDressShop is built to simulate a real-world online clothing store where users can:

- Browse fashionable dresses
- View product details
- Add products to cart
- Select size and quantity
- Login/Register
- Chat with an AI-powered FAQ chatbot
- Experience a responsive shopping interface across devices

The project demonstrates full-stack web development concepts using the MERN stack.
