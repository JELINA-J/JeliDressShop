# 👗 JeliDressShop

JeliDressShop is a **full-stack MERN e-commerce web application** designed for women's fashion. It offers a modern and responsive online shopping experience with user authentication, product browsing, shopping cart management, and an AI-powered FAQ chatbot.

---

## 📌 Project Overview

JeliDressShop simulates a real-world online clothing store where users can:

* 👗 Browse fashionable dresses
* 🔍 View product details
* 🛒 Add products to the shopping cart
* 📦 Select size and quantity before adding to the cart
* 🔐 Register and log in securely
* 🤖 Interact with an AI-powered FAQ chatbot
* 📱 Enjoy a fully responsive interface across desktop, tablet, and mobile devices

This project demonstrates practical full-stack web development using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**.
